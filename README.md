### 🗓️ 일정

- 정기 스터디: **매주 목요일 21:00 ~ (온라인)**
- 지각 규정:
    - **21:01 이후 지각**
    - **지각 인정: 당일 18:00 이전에 사전 공유한 사람만**
    - **21:05 무조건 시작**

---

### 📚 진행 규칙

1. 스터디는 **매주 목요일 오후 9시** 온라인으로 진행합니다.
2. 매주 책의 **한 챕터씩** 학습하고 배운 내용을 공유합니다.
3. 발표 범위는 챕터에 해당되는 **모든 관련 내용** 전부 가능합니다.
4. 발표자는 당일 스터디 시작과 동시에 **랜덤으로 1명** 지명합니다.
5. **무단 불참 3회**부터 스터디에서 제외됩니다.
6. 발표 자료 준비를 하지 못한 경우 **별도 패널티는 없습니다.**  
   단, 발표 미준비 시 **48시간 내 발표자료 + 요약**을 공유해야 합니다.
7. **21:01 이후 지각**으로 간주하며, **21:05 무조건 시작**합니다.

---

### ✍️ 작성 방식

- 각 챕터 자료는 **책 내용 정리**를 기본으로 작성합니다.
- 단, 단순 요약에 한정하지 않고 아래 내용을 **자유롭게 추가**할 수 있습니다.
    - 실무 경험/사례 연결
    - 관련 개념 확장(추가 학습 내용)
    - 개인적인 의견/트레이드오프 정리
    - 면접 질문 예상 Q&A
    - 궁금한 점(질문)/토론거리
    - 참고 자료 링크
